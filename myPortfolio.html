Private Clinic Management System

A real-world backend system designed to digitize and streamline the daily operations of a private medical clinic — replacing manual paper-based processes with a structured, role-based web application.

  • Problem: Private clinics manage patients, appointments, and billing manually, leading to scheduling conflicts, data loss, and poor patient experience.
  • Solution: A role-based ASP.NET Core MVC application backed by SQL Server, providing structured management for admins, doctors, and patients from a single system.
  • Role-based access control — Admin, Doctor, and Patient roles with separate dashboards
  • Full appointment scheduling: create, approve, reschedule, and cancel appointments
  • Patient medical records and treatment history persistence
  • Authentication & Authorization using ASP.NET Core Identity with JWT
  • Entity Framework Core Code-First with structured database migrations
  • Clean N-Layer Architecture: Presentation → Business Logic → Data Access
ASP.NET Core MVC Entity Framework Core SQL Server ASP.NET Identity JWT Auth N-Layer Architecture Repository Pattern C#
